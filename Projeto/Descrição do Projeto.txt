Esse projeto é um exercício de Desenvolvimento Web III e tem como finalidade  possuir três rotas e em cada exibir diferentes informações. A primeira rota é a apresentação do grupo e qual padrão GOF adotamos para o projeto. A segunda rota é um calculo da média da nota. A terceira rota utiliza a api ViaCEP para mostrar informações ao usuário sobre o CEP que o usuário digitou. Este projeto deve ser escrito em Java e fazendo o uso do framework Spring Boot.
